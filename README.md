<p>
 This is a quite old project that provides web-ui and simple json-rpc api to manage isc-dhcp server.<br>
 The backend was written in Perl, frontend - javascript/qooxdoo.<br>
</p>

## Version-1.0
 - Manage static entries
 - Search by leasesdb
 - Journal viewer
 - Config editor
 - Json-rpc api
